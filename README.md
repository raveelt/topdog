# topdog
a 2D plane dogfight

Required Modules: pygame

To play: run topdog.py

Controls:

Player 1 (red):
A Key - rotate counterclockwise
D Key - rotate clockwise
Left Shift - Shoot

Player 2 (blue): 
Left Arrow Key - rotate counterclockwise
Right Arrow Key - rotate clockwise
Right Shift - Shoot
